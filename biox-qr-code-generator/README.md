# Biox Systems QR Code Generator

## Assignment Description

This Assignment is a Python-based QR Code generator application developed for Biox Systems. 
The application accepts a URL as input and generates a QR Code image in PNG format.

## Example URL Used

https://www.bioxsystems.com/

## Tools and Technologies

- Python
- qrcode library
- Pillow library
- Visual Studio Code
- GitHub

## Assignment Files

- `main.py`: Python source code for the QR Code generator.
- `requirements.txt`: Manifest file containing required Python packages.
- `biox_qr_code.png`: Generated QR Code image.
- `application_output_screenshot.png`: Screenshot showing the expected application output.

## Installation Instructions

Install the required library using the following command:

```bash
pip install -r requirements.txt
```

## How to Run the Application

Run the Python file using the following command:

```bash
python main.py
```

When prompted, enter the following URL:

```text
https://www.bioxsystems.com/
```


## Coding Best Practices Followed

- Clear and meaningful function names
- Comments and docstrings
- URL input validation
- Organized and modular code structure
- Simple and readable program output
